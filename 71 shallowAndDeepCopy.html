->Shallow Copy:
       A shallow copy creates a new object or array, but it only duplicates the top-level properties. 
       If the properties are references (like objects or arrays), the references are copied, not the actual values.
       This means changes to nested objects or arrays will affect both the original and the copy.

    ->Deep Copy:
        A deep copy creates a new object or array, duplicating all levels of nested structures. 
        Changes to the copy will not affect the original and vice versa.

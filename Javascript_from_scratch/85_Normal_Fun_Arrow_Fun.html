Normal Function vs Arrow Function:
        ->The main difference is that arrow functions don’t have their own this, arguments, or prototype, while normal functions do. 
          Arrow functions are more concise and are mainly used for callbacks, while normal functions are better when you need dynamic context.”

        ->Key Differences:
            1)this binding (MOST IMPORTANT)
                Normal function → this depends on how it’s called
                Arrow function → inherits this from surrounding scope (lexical)

            2)Syntax:
                Arrow → shorter
                Normal → more verbose
                // Arrow
                const add = (a, b) => a + b;

                // Normal
                function add(a, b) {
                return a + b;
                }

            3)arguments object:
                Normal function → has arguments
                Arrow → does NOT have it

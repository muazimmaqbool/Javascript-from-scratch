The Map Object In JavaScript:
        A Map is a collection of keyed data items, just like an Object.
        A Map object holds key-value pairs where the keys can be any datatype.
        A Map also remembers the original insertion order of the keys.

        In Simple Terms:
        The Map object in JavaScript is a built-in object that stores key-value pairs where:
        Each key is unique.
        Keys can be any data type (not just strings like in objects).
        The order of elements is preserved (insertion order).

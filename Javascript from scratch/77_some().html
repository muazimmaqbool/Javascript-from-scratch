->The .some() method in JavaScript checks whether at least one element in an array passes a test (provided as a function).
      It returns a boolean (true or false).

    ->Syntax: 
        array.some(callback(element, index, array))
        element - The current element being processed.
        index (optional) - The index of the current element.
        array (optional) - The array the method was called on.

    ->What it does:
        Returns true if any element passes the test.
        Returns false if none pass.
        Stops iterating as soon as it finds a match (efficient).
